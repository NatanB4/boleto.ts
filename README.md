Biblioteca em Node.js para geração de boletos utilizando PDFKit 

Geração de boletos para bancos:
- Bradesco
- Caixa
- Ailos (Cecred)
- Itaú
- Sicoob
- Sicredi
- Santander
- Banco do Brasil

### Install

```javascript
npm i boletos.ts
```
```javascript
yarn add boletos.ts
```
### Exemplos de uso

```javascript
npm run gerarBrasil
```
```javascript
npm run gerarBradesco
```
```javascript
npm run gerarCecred
```

* [pasta com arquivos de exemplo](/examples)
